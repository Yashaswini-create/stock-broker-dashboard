# Stock Broker Client Web Dashboard

## Description
This project is a real-time stock broker dashboard where multiple users can log in, subscribe to stocks, and view live price updates without refreshing the page.

## Features
- Email-based login
- Subscription to fixed stocks (GOOG, TSLA, AMZN, META, NVDA)
- Random stock price generation
- Real-time asynchronous updates
- Multiple users supported simultaneously

## Tech Stack
- Node.js
- Express.js
- Socket.IO
- HTML, CSS, JavaScript

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `node server.js`
4. Open http://localhost:3000 in a browser
